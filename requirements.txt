# NUMERISCHE BIBLIOTHEKEN

# NumPy: Fundamentale Bibliothek für wissenschaftliches Rechnen
# - Schnelle Array-Operationen (C-optimiert)
# - Zufallszahlen-Generierung (Mersenne Twister)
# - Mathematische Funktionen (exp, sqrt, etc.)

# Version ≥1.20.0: Einführung von stabileren Random Number Generators
# Version ≥1.21.0: Verbesserte Typisierung für wissenschaftliche Anwendungen

# ============================================================================
# BILDVERARBEITUNG
# ============================================================================

# - 16-bit Integer Support (uint16 für realistische Photon Counts)
# - Kompression (tiff_deflate)
#
# Version ≥9.0.0: Sicherheitsupdates und bessere TIFF-Unterstützung
# Version ≥9.2.0: Verbesserte Performance für große TIFFs
Pillow>=9.2.0

# ============================================================================
# VISUALISIERUNG (Optional, aber empfohlen)
# ============================================================================

# Matplotlib: Plotting und Visualisierung
# - Wird für optionale Datenvisualisierung verwendet

#
# Version ≥3.5.0: Verbessertes Layout-System
# Version ≥3.6.0: Bessere 3D-Plotting-Funktionen

# ============================================================================
# PROGRESS BARS
# ============================================================================
